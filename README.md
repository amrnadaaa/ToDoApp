# ToDoApp 

This is a small project where I applied **JDBC** to connect with a database.  
It demonstrates simple **CRUD operations** (Create, Read, Update, Delete) using **Java**.

## Features
- Connect to database using JDBC
- Add new tasks
- View tasks
- Update existing tasks
- Delete tasks

## Technologies Used
- Java
- JDBC
- IntelliJ IDEA
- Git & GitHub
